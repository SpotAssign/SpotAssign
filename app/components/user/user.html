Welcome to Spot Assign!

There are a few things that you should know to get started.
  1. To create an event, click "Create Event" and fill out the necessary fields.
  2. When you create an event you will be taken to a different dashboard where you will be able to access all your event information including user, payment, and map information.
  3. To find events or event information you can use the "Find Event" page.
